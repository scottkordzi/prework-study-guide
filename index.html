Prework Study Guide

HTML

  • HTML notes such as...
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Section elements are semantic and define a section in a document that contains thematic content.

CSS

  • CSS notes such as...
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Width indicates how large or small the content is from left to right.
  • Length indicates how large or small the content is from top to bottom.
  • The img selector assigns property values to any image element.
  • The display property is assigned with the block value, which assigns each image to start on a new line.

GIT

  • GIT notes such as...
  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • git add [file]: adds a file as it looks now to your next commit (stage).
  • git commit -m "[descriptive message]": commits your staged content as a new commit snapshot.
